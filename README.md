# Offline Customer Support Chatbot

A functional offline customer support chatbot built using **Ollama** and Meta's **Llama 3.2 3B** model. This project demonstrates local LLM deployment for privacy-conscious e-commerce applications.

## Key Features
- **Local Inference**: Runs completely offline using Ollama, ensuring data privacy.
- **Prompt Engineering**: Compares Zero-Shot vs. One-Shot prompting techniques.
- **Automated Evaluation**: Processes 20 adapted queries and logs responses for manual scoring.

## Getting Started
Refer to [setup.md](setup.md) for installation and execution instructions.

## Evaluation Summary
Our evaluation showed that **Zero-Shot** prompting provided more robust and helpful responses than One-Shot for this specific model and dataset. Detailed scores and qualitative analysis can be found in [report.md](report.md).
